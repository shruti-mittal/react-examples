# ReactJS Examples

### Prerequisites
* php v5.6
* [node-js](https://github.com/creationix/nvm) >=10.11 (with npm installed)

### Setup
* Clone this repository, switch to active branch.
* Install dependencies
```
npm install
```

* Start development
```
npm start
```
Or
```
npm run dev
```

* Can view changes on http://localhost:3000/ using webpack-dev-server
